CLI reference

All neural commands.

Quick start
# Scaffold a config neural init --model mlp --name my_exp # Train neural train --config runs/my_exp/config.yaml # Evaluate neural evaluate --config runs/my_exp/config.yaml # Serve as REST API neural serve --config runs/my_exp/config.yaml # This dashboard neural dashboard
neural train
# Override hyperparams neural train -c config.yaml \ -O training.num_epochs=100 \ -O training.batch_size=128 \ -O training.optimizer.lr=5e-4 # Custom DB path neural train -c config.yaml --db /tmp/forge.db
neural serve
neural serve -c config.yaml \ -k runs/my_exp/checkpoints/checkpoint_best.pt \ --host 0.0.0.0 --port 8080 # Endpoints GET /health GET /info POST /predict GET /docs — Swagger UI
neural export
# Export to ONNX neural export -c config.yaml --format onnx # Export to TorchScript neural export -c config.yaml --format torchscript
